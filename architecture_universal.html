Architecture Components

  • Experience Layer: The user interface integrated directly into the Matrix Portal.
  • Intelligence Layer: The Node.js engine that translates English into secure, verified SQL.
  • Reasoning Layer: The 2026 Cloud Model Pool (Llama 4 / Qwen 3) that provides the "brain" for complex queries.
  • Persistence Layer: Your local COSEC_DEMO database providing real-time facts.

Operational Use Cases

Attendance Audit: "Who was late yesterday?" -> Identifies punch gaps using Llama 4 reasoning.
Canteen Tracking: "Total subsidy spent today?" -> Aggregates canteen logs across all terminals.
